- [ ] Open Product Hunt in browser
- [ ] Log in and verify session
- [ ] Copy first comment to clipboard

- [ ] Notifications enabled (phone, desktop)
- [ ] Team standup call (if applicable)

### 11:45 PM PST

- [ ] Product Hunt submission draft opened
- [ ] First comment in clipboard (ready to paste)
- [ ] Social posts queued
- [ ] Email blast queued
- [ ] Final deep breath 🧘

### 12:01 AM PST - 🚀 GO LIVE!

**IMMEDIATELY:**

- [ ] Submit product to Product Hunt
- [ ] Post first comment (< 60 seconds!) ⏰
- [ ] Screenshot submission confirmation
- [ ] Tweet: "We're live on PH! [link]"
- [ ] Send email blast to waitlist
- [ ] Post on LinkedIn
- [ ] Post in Indie Hackers
- [ ] Post in Discord/Slack communities

**Within 5 Minutes:**

- [ ] Respond to first comments

- [ ] Respond to EVERY comment (< 5 min response time)
